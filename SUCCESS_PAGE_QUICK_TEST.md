## 🐛 Common Issues

1. **Page doesn't load**: Check route is configured
2. **Sections missing**: Check component render logic
3. **Data shows "undefined"**: Check mock data structure
4. **Countdown doesn't work**: Check useEffect dependencies
5. **Buttons don't navigate**: Check navigate paths
6. **Pro tip always shows**: Check `source === 'hrms'` condition
7. **Score bar wrong**: Check `getScoreBarFill()` calculation
