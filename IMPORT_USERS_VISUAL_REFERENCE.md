## Responsive Behavior

### Desktop (1024px+)
- Modal width: 768px (max-w-3xl)
- Three columns possible in footer
- Full button text visible

### Tablet (768px - 1023px)
- Modal width: 90vw
- Two columns in footer
- Full functionality maintained

### Mobile (< 768px)
- Modal width: 95vw
- Stacked buttons
- Smaller padding
- Same functionality, adapted layout

---

## Animation & Transitions

### Modal Open
- Fade in: 200ms
- Slide up: 20px
- Overlay: Opacity 0 → 0.75

### Modal Close
- Fade out: 150ms
- Shrink: Scale 1 → 0.95

### File Upload
- Drop zone hover: Border color transition 200ms
- Processing: Spinner rotation continuous

### Buttons
- Hover: Background color 150ms
- Active: Scale 0.98

```css

  - Expand all scrollable areas
```
