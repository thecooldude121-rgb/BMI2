## If Something Doesn't Work

**Navigation not working?**
- Check console for errors
- Verify route is added to CRMModule.tsx
- Ensure meeting ID is valid

**Styling looks wrong?**
- Check inline styles are applied
- Verify Tailwind classes correct
- Check for CSS conflicts

**Action items not toggling?**
- Check state updates in console
- Verify toast context loaded
- Check onClick handlers

**Sidebar not sticky?**
- Check "position: sticky" is applied
- Verify "top: 2rem" is set
- Check parent container has no overflow hidden
