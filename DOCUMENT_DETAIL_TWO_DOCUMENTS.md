## Special Badges

### HRMS Badge Features

### Visual Design
**Hero Section Badge**:
- Emerald green background (bg-emerald-100)
- Emerald text (text-emerald-800)
- Border for emphasis (border-emerald-200)
- Link icon (🔗) prefix
- Text: "HRMS Connected"
- Appears after Version badge

**Details Card Section**:
- Circular emerald icon with link symbol
- Bold title: "HRMS Connected"
- Subtitle: "Synced with employee records"
- Separated section with border-top
- Professional, clean appearance

### Conditional Display
The HRMS badge only appears when:
```typescript
document.hrms_connected === true
```

For the TechStart document, this badge indicates the document is connected to employee records in the HRMS system.

## Color Systems

### HRMS Badge Color System

**Emerald Green Palette**:
- Background: `bg-emerald-100` (#d1fae5)
- Text: `text-emerald-800` (#065f46)
- Border: `border-emerald-200` (#a7f3d0)
- Icon BG: `bg-emerald-100` (#d1fae5)

**Why Emerald?**
- Distinct from other badges (blue, gray)
- Represents connection/integration
- Professional and modern
- Good contrast and readability
- Associated with "verified" or "connected" status

### AI Generated Badge Color System

**Purple Palette**:
- Background: `bg-purple-100` (#f3e8ff)
- Text: `text-purple-800` (#6b21a8)
- Border: `border-purple-200` (#e9d5ff)
- Icon BG: `bg-purple-100` (#f3e8ff)

**Why Purple?**
- Distinct from HRMS (emerald) and standard badges (blue, gray)
- Associated with AI, innovation, and technology
- Professional and modern appearance
- Excellent contrast and readability
- Represents automation and intelligence
- Not the typical indigo/violet (following design requirements)

### Share Document Functionality

**How It Works**:
1. Click "Share" button in document header
2. Select team member from dropdown (shows name and email)
3. Choose visibility level (Private/Team/Company)
4. Add optional message
5. Click "Share Doc"

**What Happens**:
- Team member added to "Shared With" section
- Activity log entry created: "Alex Rodriguez shared with [Member Name]"
- Success toast notification displayed
- Shared user appears in details card

**Example**:
```
Share with: Emily Davis (emily.davis@bmi.com)
Visibility: Team
Message: "Please review this proposal"
```

Result: Emily Davis appears in shared users list with timestamp.

## Activity Log Tracking

All interactions are automatically logged with:
- **User**: Alex Rodriguez (currentUser.user_name)
- **Timestamp**: Current date/time
- **Action Type**: view, download, share, edit, delete
- **Icon**: Color-coded based on action type
  - View: Blue eye icon
  - Download: Green download icon
  - Share: Purple share icon
  - Edit: Yellow edit icon

**Appears In**:
- Activity Timeline tab in main content area
- Latest activity shows at top of list
