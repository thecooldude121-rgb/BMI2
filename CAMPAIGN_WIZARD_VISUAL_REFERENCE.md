## COLOR SCHEME

**Primary Actions:**
- Blue (#3B82F6) - Next buttons, current step, primary actions
- Green (#10B981) - Completed steps, Launch Campaign button
- Gray (#6B7280) - Disabled states, inactive elements

**Status Indicators:**
- Green with checkmark - Completed
- Blue circle - Current/Active
- Gray circle - Not started/Inactive

**Backgrounds:**
- White (#FFFFFF) - Main content areas
- Gray 50 (#F9FAFB) - Page background
- Blue 50 (#EFF6FF) - Info panels
- Gray 100 (#F3F4F6) - Disabled buttons

**Borders:**
- Gray 200 (#E5E7EB) - Default borders
- Blue 300 (#93C5FD) - Hover states
- Blue 500 (#3B82F6) - Selected states

## RESPONSIVE BEHAVIOR

**Desktop (1024px+):**
- Full 6-column template grid
- Side-by-side navigation buttons
- Expanded progress tracker

**Tablet (768px - 1023px):**
- 3-column template grid (2 rows)
- Stacked navigation in some cases
- Compressed progress tracker

**Mobile (< 768px):**
- Single column template grid
- Stacked navigation buttons
- Simplified progress tracker
- Scrollable content areas

---

## INTERACTIVE ELEMENTS

**Hover Effects:**
- Template cards: Border changes to blue, subtle shadow
- Buttons: Darker background shade
- Toggle switches: Slight scale up

**Click Feedback:**
- Template selection: Border turns blue, card highlights
- Navigation: Button press effect
- Toggle switches: Smooth slide animation

**Focus States:**
- Input fields: Blue ring appears
- Buttons: Blue outline
- Template cards: Blue border
